# 使用__name__对模块进行测试

代码在当前文件能执行，被被的代码导入的时候不执行：
用：
`if __name__ = '__main__':`  # 此代码只会在本代码中执行

# 魔术变量：`__name__`
- 随着运行环境的不同，返回的结果也不同（结果是字符串）
- 当被导入是`__name__`是本页面的**文件**名称，不等于`__main__`
