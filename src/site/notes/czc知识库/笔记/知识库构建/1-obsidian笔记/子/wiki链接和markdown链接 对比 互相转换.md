# wiki链接 

1. 文件链接：`[[link|name]]`
2. 图片链接：`![[link|name]]` 
3. 外链：没有外链
# markdown链接

1. 文件链接：`[name](link)`
2. 图片链接：`![name](link)`
3. 外链：`[site](https://www.xxx.com/)`
