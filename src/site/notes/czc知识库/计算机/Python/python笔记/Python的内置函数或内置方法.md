> 字符串的内置方法
> ```python
> class Drebin:
> 	def __init__(self, random_state=2020, run_id="default", data_path = '', working_path="./"):   
> 		if not working_path.endswith('/'):  
> 			working_path += '/'
> ```
> `working_path.endswith('/')`这个endswith是字符串的内置方法，只要对象是字符串，就可以使用字符串的内置方法。
