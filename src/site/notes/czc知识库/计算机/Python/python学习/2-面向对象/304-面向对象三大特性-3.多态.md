# 三种特性：
- **封装性**：
	- 私有属性和私有方法
	- 把属性和方法隐藏起来，属性没有特殊要求都用私有属性
	- **意义**：
		- 以面向对象的编程思想进行项目开发
		- 封装数据属性：明确的区分内外，控制外部对隐藏的属性的操作行为(保护数据、过滤掉异常数据)
- **继承性**：
	- 单继承和多继承
- **多态性**

属性和方法的封装

# 3、多态

## 什么是多态（定义）

定义：多态是一种使用对象的方式，子类重写父类方法，调用不同子类对象的相同父类方法，可以产生不同的执行结果
①多态依赖继承
②子类方法必须要重写父类方法

好处：调用灵活，有了多态，更容易编写出通用的代码，做出通用的编程，以适应需求的不断变化！
